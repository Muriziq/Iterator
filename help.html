Help & Reference

Everything you need to know about using the Iterator. Browse by topic or search for a specific tool or shortcut.

Tools

Toolbar
Move Tool Selects and moves individual objects on the canvas. Click an object to select it, then drag to reposition it.
Multiple Selection Tool Selects multiple objects at once by clicking each one or dragging a selection box around them. All selected objects can be moved together.
Pan Tool Pans and navigates around the canvas without selecting or modifying any objects. Click and drag to scroll the view in any direction.
Add Rectangle Tool Draws a rectangle on the canvas. Click and drag to define the width and height of the shape.
Add Ellipse Tool Draws an ellipse or circle on the canvas. Click and drag to define the size of the shape.
Add Line Tool Draws a straight line on the canvas. Click to set the start point, then drag to define the direction and length.
Add Polygon Tool Draws a polygon on the canvas. You can configure the number of sides in the properties panel before or after drawing.
Add Image Tool Places an image on the canvas from your device. Once added, the image can be resized, repositioned, and used as an iteration source.
Add Textbox Tool Adds a text box to the canvas. Type directly into it and style the text using the properties panel. Text boxes can also be iterated with dynamic content.
Zoom Tool Zooms in and out of the canvas. Click to zoom in, hold Alt and click to zoom out, or use the scroll wheel for fine control.
Duplicate Tool Creates an exact copy of the currently selected object and places it slightly offset from the original. Works with single objects and groups.
Delete Tool Permanently removes the currently selected object or group from the canvas. This action can be undone with Ctrl + Z.
Layer Order Tool Controls the stacking order of objects on the canvas. Use it to bring objects forward, send them backward, or move them to the very front or back of the layer stack.
Align Top Aligns the selected object to the top edge of the canvas. When multiple objects are selected, they are all aligned to the top edge of the last object selected.
Align Center (Vertical) Aligns selected objects to the vertical center of the canvas or to each other's vertical midpoint when multiple objects are selected.
Align Bottom Aligns the selected object to the bottom edge of the canvas. When multiple objects are selected, they are all aligned to the bottom edge of the last object selected.
Align Left Aligns the selected object to the left edge of the canvas. When multiple objects are selected, they are all aligned to the leftmost edge among them.
Align Center (Horizontal) Aligns selected objects to the horizontal center of the canvas or to each other's horizontal midpoint when multiple objects are selected.
Align Right Aligns the selected object to the right edge of the canvas. When multiple objects are selected, they are all aligned to the rightmost edge among them.
Flip Horizontal Mirrors the selected object horizontally, producing a left-to-right reflection. Works on single objects and groups.
Flip Vertical Mirrors the selected object vertically, producing a top-to-bottom reflection. Works on single objects and groups.
Group Tool Combines two or more selected objects into a single group. Grouped objects can be moved, resized, and transformed together as one unit.
Clip Tool Clips the selected objects so that only the portion within the topmost shape (the clipping path) remains visible. Everything outside the clipping path is hidden.

Shape Tools

Edit Mode
These tools are available when an object is in Edit Mode. Switch to Edit Mode first to access shape-editing controls.
Convert to Curve Converts a straight line segment into a smooth Bézier curve. Drag the control handles that appear to adjust the curve's shape freely.
Convert to Rounded Edge Converts a curve segment into a rounded corner with a uniform radius, giving it a smooth, consistent arc rather than a freeform curve.
Edit Mode Switches the selected object into Edit Mode, revealing its individual anchor points and segments so you can reshape it freely.
Normal Mode Returns the object from Edit Mode back to Normal Mode. In Normal Mode you can resize, move, rotate, and interact with the object alongside other objects on the canvas.

Rectangle Tools

Rectangle Only
These tools only appear in the properties panel when a rectangle is selected.
Bevel Tool Adds a bevel effect to the corners of a selected rectangle, cutting each corner at a diagonal angle. Adjust the bevel size in the properties panel.
Round Tool Adds a rounded corner effect to a selected rectangle. Increasing the value produces softer, more circular corners.
Shape Tool Breaks the selected object down into its individual line segments, each of which can then be repositioned and edited independently.

Text Iteration

Textbox
Format Iterated Controls how iterated text that exceeds the original text box width is displayed. Use this when your data contains entries longer than the original text.
Iterate Align Sets how all iterated text entries are aligned and how overflow is handled. The available modes are:

None — No adjustment. Text renders exactly as-is with no overflow handling.
Fit — All text entries attempt to fit within the width of the original text box, scaling as needed.
Shrink to Fit — Only entries longer than the original shrink to fit; shorter entries are unaffected.
Create New Line — Text longer than the original width wraps onto a new line automatically.
At White Space — Text wraps onto a new line only at word boundaries (spaces), preventing mid-word breaks.

Image Options

Image
Aspect Ratio Locks the width-to-height ratio of the original image file when resizing. Enable this to prevent distortion when changing the image dimensions.
Add Images Adds more images to the iteration sequence. Each image becomes a separate output. Three sizing modes are available:

Maintain Size — Preserves the exact width and height of the original image for every added image.
Maintain Width — Keeps the original width fixed; the height adjusts proportionally to match each added image's aspect ratio.
Maintain Height — Keeps the original height fixed; the width adjusts proportionally to match each added image's aspect ratio.

Generate

Output
Generate Button Starts the iteration process, rendering all outputs based on your current canvas design and data. Once generation is complete, the results are ready to be exported as a PDF or image file.

Keyboard Shortcuts

Quick Reference
↑ Arrow Move selected object up. If nothing is selected, pans the canvas up.
↓ Arrow Move selected object down. If nothing is selected, pans the canvas down.
← Arrow Move selected object left. If nothing is selected, pans the canvas left.
→ Arrow Move selected object right. If nothing is selected, pans the canvas right.
Ctrl + D Duplicate the selected object.
Shift + @ Zoom to fit the selected object in view.
Ctrl + A Select all objects on the canvas.
Ctrl + G Group selected objects (minimum 2 required).
Ctrl + S Save the project to file.
Ctrl + U Ungroup the selected group.
Delete Delete the selected object.
Page Up Move selected object one step forward in layer order.
Page Down Move selected object one step back in layer order.
Home Bring selected object to the front.
End Send selected object to the back.
L Align selected objects to the left.
C Align selected objects to the horizontal center.
R Align selected objects to the right.
T Align selected objects to the top.
E Align selected objects to the vertical center.
B Align selected objects to the bottom.
Q Flip selected objects horizontally.
W Flip selected objects vertically.
M Switch to Move Tool.
S Switch to Multiple Selection Tool.
H Switch to Pan Tool.
Ctrl + Z Undo the last action.
Ctrl + Y Redo the last undone action.
Z Switch to Zoom Tool.
P Switch to Add Line Tool.
